# Virtual Power Plant

Virtual Power Plant is a Spring Boot demonstration application.

## Prerequisites

Install [openJdk 11](https://www.openlogic.com/openjdk-downloads)

#Build
```bash
mvn install
```

## Run
```bash
java -jar ../virtualpowerplant-0.0.1-SNAPSHOT.jar.jar
```

## API Documentation
